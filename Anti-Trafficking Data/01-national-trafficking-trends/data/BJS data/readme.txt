Human Trafficking Data Collection Activities, 2024   NCJ 309422
	
This zip archive contains tables in individual  .csv spreadsheets	
from Human Trafficking Data Collection Activities, 2024   NCJ 309422.  The full report including text	
and graphics in pdf format is available from: https://bjs.ojp.gov/library/publications/human-trafficking-data-collection-activities-2024
	
This report is one in a series.  More recent editions	
may be available.  To view a list of all in the series go to:	
https://bjs.ojp.gov/library/publications/list?series_filter=Human%20Trafficking%20Data%20Collection%20Activities
